# AREPargne

AREPargne est un guide interactif dédié aux finances personnelles. Le site propose des articles clairs sur l'épargne, les investissements et la préparation de la retraite.

## 🎯 Objectifs

- Aider chacun à comprendre les bases de la gestion financière.
- Présenter les différents véhicules d'épargne et d'investissement.
- Offrir des ressources pour planifier sa retraite et définir ses objectifs financiers.

## 🚀 Instructions de développement

Prérequis : Node.js 18+.

```bash
npm install          # Installe les dépendances
npm run dev          # Lance le serveur de développement sur http://localhost:4321
npm run build        # Compile le site pour la production dans ./dist
npm run preview      # Prévisualise la version produite
```

## 🌐 Déploiement

Le site peut être déployé sur n'importe quelle plateforme statique (Vercel, Netlify, GitHub Pages…).
Lien de déploiement : https://arepargne.example.com
